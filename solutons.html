Membuat WEBSITE Dengan Mengandalkan Bahasa Pemrograman Sendiri

Untuk membuat website.com Anda harus benar-benar mempunyai pengetahuan pemrograman, karena kalau hanya menguasai HTML maka hasilnya bisa jadi website yang menyedihkan dalam arti yang sesungguhnya.


Kalau Anda ingin membuat website dari file yang bisa di hosting, maka setidaknya Anda harus menguasai 3 hal:

*Bahasa pemrograman tertentu seperti PHP

*Penggunaan software Editor File Web

*Pengelolaan database seperti MySQL


Belajar Bahasa Pemrograman Untuk Membuat Website

Kalau Anda mau belajar, maka sebaiknya Anda membeli satu buku panduan di toko buku tentang PHP dan MySQL. Atau kalau Anda mau belajar sendiri Anda bisa mencoba website yang luar biasa W3SCHOOL (harus ada dasar bahasa inggris yang cukup). Lihat menu pembelajarannya di sidebar sebelah kiri.


Belajar Program Pembuat Website Seperti DREAMWEAVER

Menguasai bahasa pemrograman saja tidak berarti Anda bisa membuat websitenya tanpa bantuan, jadi Anda perlu belajar cara membuat website dengan menggunakan software. Salah satu software yang cukup umum digunakan adalah Dreamweaver.


Anda bisa membeli buku panduan Adobe Dreamweaver di toko-toko buku terdekat, karena panduan menggunaka buku jauh lebih terstruktur. Kalupun Anda harus belajar dari dasar, tidak akan terlalu sulit. Dlam jangka waktu sekitar 2 mionggu Anda sudah bisa merampungkan Website PHP sederhana milik Anda....


Tapi kalau masalahnya ada di ekonomi yang lagi seret buat beli buku.... Bisa coba baca tutorial e-book di http://www.bgsu.edu/downloads/cio/file17760. pdf .... tapi bahasa inggris ya....
